# P300 Workspace

This workspace contains several Jupyter Notebooks for modeling and analysis related to P300 data. Below is an overview of the included files:

## Notebooks
- **Hold_Out_models.ipynb**: Contains models and analysis using a hold-out validation approach.
- **KFold_models.ipynb**: Implements k-fold cross-validation models for robust evaluation.
- **Loso_models.ipynb**: Features leave-one-subject-out (LOSO) cross-validation models.
- **SATS-Net.ipynb**: Notebook for SATS-Net, a specialized neural network model for P300 data.

## Getting Started
1. Open any of the notebooks in VS Code or JupyterLab.
2. Ensure you have the required Python environment and dependencies installed (e.g., numpy, pandas, scikit-learn, tensorflow/keras, matplotlib, etc.).
3. Run the cells in the notebook to reproduce the analyses or train models.

## Notes
- Each notebook is self-contained and includes code, results, and explanations.
- For best results, use a Python 3.8+ environment with the necessary scientific libraries installed.
